Học lập trình website


MỤC 1 — NỀN TẢNG WEB (Frontend Basic)

Chương 1 — HTML cơ bản

Bài 1: Cấu trúc HTML

Kiến thức:

  • !DOCTYPE html
  • html, head, body
  • thẻ mở / đóng
  • comment HTML
Ví dụ: 
          <!DOCTYPE html>
          <html>
          <head>
              <title>Trang web</title>
          </head>

          <body>

          </body>
          </html>

Bài 2: Thẻ văn bản

Kiến thức:

  • "h1 → h6" : thẻ tiêu đề
  • "p" : thẻ đoạn văn
  • "br" : thẻ xuống dòng
  • "b", "i", "u"
  • danh sách: "ul", "ol"

Bài 3: Ảnh + Link

Kiến thức:

  • "img"
  • src
  • alt
  • a href=""
  • đường dẫn tương đối / đường dẫn tuyệt đối

Chương 2 — CSS cơ bản

Bài 1: CSS là gì ?

Kiến thức:

  • inline CSS
  • internal CSS
  • external CSS
  • selector
Ví dụ: 
  h1 {
       color: red;
     }

Bài 2: Màu sắc + chữ

Kiến thức:

  • color
  • font-size
  • font-family
  • font-weight
  • text-align

Bài 3: Border + Box Model

Kiến thức:

  • border
  • padding
  • margin
  • width
  • height

Rất quan trọng vì đây là nền tảng layout.

Chương 3 — Layout cơ bản

Bài 1: Display

Kiến thức:

  • block
  • inline
  • inline-block

Bài 2: Flexbox

Kiến thức:

  • display:flex
  • justify-content
  • align-items
  • flex direction

Đây là kỹ năng cực kỳ quan trọng.

Bài 3: Responsive cơ bản

Kiến thức:

  • max-width
  • %
  • vh/vw
  • media query
